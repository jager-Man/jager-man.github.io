对yum源的配置

把yum源理解为安装工具就行了

配置

1.备份系统默认的yum源
cd到该目录

cd/etc/yum.repos.d/

2.创建文件夹备份

mkdir bak

mv *.repo bak

怎么配置

  1. 挂载光盘(临时)


2. 配置源文件

  • name:随意命名
  • baseurl:file(本地源,已经挂载到cdrom目录)
  • http,https(网络源)
  • gpgcheck:密钥检测

apache

1.安装apache

yum install httpd httpd-devel

2.启动apache服务和一些相关命令

1.启动

systemctl start httpd

2.停止 / 重启

systemctl stop/restart httpd

3.查看状态

systemctl status httpd

4.设置httpd服务开机启动

systemctl enable |httpd

5.关闭防火墙
6.查看状态
